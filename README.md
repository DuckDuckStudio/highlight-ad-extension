## 如何获取
你可以参照代码文档手动加载扩展，也可以从[Edge加载项](https://microsoftedge.microsoft.com/addons/detail/%E9%AB%98%E4%BA%AE%E2%80%9C%E5%B9%BF%E5%91%8A%E2%80%9D/ogpplfibejcooaoooohiphklgfoabhie)获取扩展。
